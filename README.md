# Tic Tac Toe Game 🎮

A simple and fun **Tic Tac Toe** game built with **HTML**, **CSS**, and **JavaScript**. Challenge your friends to see who claims victory in this classic game!

## Features
- 🕹️ **Two-player mode**: Play with a friend on the same device.
- 🎨 **Responsive Design**: Optimized for desktops and mobile devices.
- 🏆 **Interactive UI**: Smooth animations and user-friendly interface.

## How to Play
1. Clone or download the repository.
2. Open the `index.html` file in your browser.
3. Start playing and enjoy!

## Technologies Used
- **HTML** for structure.
- **CSS** for styling.
- **JavaScript** for game logic.

## File Structure
```
|-- index.html   # Main HTML file
|-- style.css    # Stylesheet for the game
|-- script.js    # JavaScript for game logic
```
